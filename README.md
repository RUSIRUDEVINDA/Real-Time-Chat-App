# 💬 Real-Time Chat App

A simple **real-time chat application** built using **Node.js, Express, and Socket.IO**.  
The application uses **WebSockets** to create a persistent connection between users, allowing messages to be sent and received instantly without reloading the page.

---

## 🧩 What We’re Building (Simple Version)

A web page with:

- A username input
- A message input
- A chat area

### Multiple users can:

- Join the chat
- Send messages
- See messages instantly

---

## 🛠️ Tech Stack

- Node.js
- Express
- Socket.IO
- Plain HTML + JavaScript (No frameworks)

---

## 🪜 Step-by-Step Roadmap

1. Set up development tools
2. Create a basic server
3. Display a web page
4. Integrate Socket.IO
5. Send messages
6. Display messages in real time
