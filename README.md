# Real-Time-Chat-App

This is a simple real-time chat application built using Node.js, Express, and Socket.IO. The server creates a persistent connection between users using WebSockets, which allows messages to be sent and received instantly without reloading the page

🧩 What We’re Building (Simple Version)

.A web page with:
A username input
A message input
A chat area

Multiple users can:
Join
Send messages
See messages instantly

-----Tech-----
Node.js
Express
Socket.IO
Plain HTML + JavaScript (no frameworks)
